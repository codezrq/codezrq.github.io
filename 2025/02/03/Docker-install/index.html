二. 配置用户组

默认情况下，只有root用户和docker组的用户才能运行Docker命令。我们可以将当前用户添加到docker组，以避免每次使用Docker时都需要使用sudo。

如果出现“启动“docker.service”需要认证。Multiple identities can be used for authentication:”的报错，说明是没有将当前用户加入到docker用户组中。
